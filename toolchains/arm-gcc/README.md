# ARM GCC 工具链集成说明

## 关于 ARM GCC

ARM GCC (GNU Arm Embedded Toolchain) 是用于 ARM Cortex-M 和 Cortex-R 处理器的免费开源工具链。

- **版本**: 10.3.1
- **许可证**: GNU General Public License v3.0 (GPLv3)
- **官方网站**: https://developer.arm.com/tools-and-software/open-source-software/developer-tools/gnu-toolchain

## 使用方法

详细的使用说明请参阅项目文档：`docs/toolchain-integration.md`
